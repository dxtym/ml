## Usage

1. Clone the repository:
```bash
git clone git@github.com:dxtym/ml.git 
cd ml
```

2. Set up virtual environment:
```bash
pip install uv
uv venv
source .venv/bin/activate
```

3. Run the app locally:
```bash
make run
```

## Achievements:

- Logistic Regression: 92% (ACC) and 91% (F1)
- Random Forest: 88% (ACC) and 87% (F1)
- XGBoost: 88% (ACC) and 86% (F1)
